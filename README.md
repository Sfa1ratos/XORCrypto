# MinGwTest

Первое окно открывается для визуального отображения, что программа не загружает основной поток, а все изменения файлов проходят в отдельном потоке.

Окно с настройками открывается по кнопке "Шифрование"

Ввод маски позволяет добавить в выпадающий список любые маски, по мимо стандартных.
